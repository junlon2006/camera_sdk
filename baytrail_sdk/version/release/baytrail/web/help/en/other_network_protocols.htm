DDNS                               Return to Contents

To enable the Dynamic Domain Name System (DDNS):

1. Select the Enable check box.

2. Specify parameters displayed.

3. Click Save.

Note: Before enabling the DDNS, register with the target DDNS server provider and configure DNS settings and the default gateway.
PPPoE                               
To enable PPPoE:

1. Select the Enable check box.

2. Specify User name and Password.

3. Click Save.

K-SNMP                               

Kedacom Platform Management Client (PMC) uses the K-SNMP to manage cameras.

To configure K-SNMP settings:

1. Configure parameters displayed.

2. Click Save.

802.1X                               

The IEEE 802.1X is an IEEE standard for Port-based Network Access Control (PNAC). The IEEE 802.1X is a networking protocol and it provides an authentication mechanism for devices that want to attach to a LAN or WLAN.

To configure the IEEE 802.1X:

1. Select the Enable check box.

2. Configure parameters displayed.

3. Click Save.

QoS                               

To configure the QoS:

1. Select the Enable check box.

2. Configure parameters displayed. Note that the value 0 indicates the lowest priority level and the value 63 the highest priority level.

3. Click Save.

UPnP                               

To enable the Universal Plug and Play (UPnP) technology:

1. Click the Enable check box.

2. Enter an alias for the camera.

3. Select a mapping mode.

If you choose the auto mode, you do not need to configure the mapping port table.

If you choose the manual mode, you need to configure the mapping port table.

4. Click Save.
