On the Local tab page, you can configure local settings.

Play
Protocol The default value is UDP.
Performance This parameter indicates the playing performance. The default value is Real-time.
Enable Image Denoising Whether to enable the Image Denoising function. When this check box is selected, the function takes effect at the local end. Additionally, you need to select a denoising level. Level 4 is the highest level.
Note: A high denoising level may cause blurred edges for moving objects.
Enable V-Sync Whether to enable the Vertical Synchronization function.
Note: Enabling this function will increase the CPU usage.
Recording
Packet Size Size of each record packet. The default value is 512M.
Save Path Path where local recordings are saved. The default value is C:\IPCWeb\Rec.
Clip Save Path Path where clips are saved. The default value is C:\IPCWeb\Video\PlayBackCut. Note that you can clip a record during playback.
Download Recording
Save Path If you download a recording during a playback, the recording is saved in the path determined by this parameter. The default value is C:\IPCWeb\RecPlayback.
Snapshot
Previewing: snapshot save path Path where snapshots taken during live view are saved. The default value is C:\IPCWeb\Pic\Preview.
Camera: snapshot save path Path where snapshots downloaded from the Snapshot tab page are saved. The default value is C:\IPCWeb\Pic\PuGrab.
Playback: snapshot save path Path where snapshots taken during playback are saved. The default value is C:\IPCWeb\Pic\PlayBack.
