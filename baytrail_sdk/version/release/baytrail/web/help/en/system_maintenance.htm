You can use the flash function to find your camera when you cannot find it.

To use the flash function, specify Flash time and click Flash.
Note: The flash function is available only to cameras equipped with an LED indicator RUN.
The Auto Maintenance function allows you to restart the camera at a scheduled time.
To use the Auto Maintenance function:

1. Select the Enable check box.

2. Specify Restart On and Restart At.

3. Click Save.

You can use *.pkg files to upgrade the camera. During an upgrade, do not perform any operation. After the upgrade, log in to the camera again.
The Partial Restoration parameter indicates that factory defaults apply to all parameters except the camera's IP address and access protocol settings.
The Complete Restoration parameter indicates that factory defaults apply to all parameters.
