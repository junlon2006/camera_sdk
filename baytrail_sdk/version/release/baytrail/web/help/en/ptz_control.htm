PTZ Control                                Return to Contents

For bullet and box cameras, PTZ controls are hid by default. To show PTZ controls, click the expand button on the Live View tab page.

The following table provides the description for each PTZ control.

Direction Arrow Each arrow indicates a direction.
Reset/Patrol

Click to enable a bullet or box camera to go to the factory default point and zoom out.

Click to enable a dome camera to start a pan patrol.

Step A large step indicates a high PT speed. The step adjustment bar can also control the zooming and focusing speeds for cameras with a motorized zoom lens.
Zoom Click to zoom in and to zoom out.
Focus Click for far focus, for near focus, and for autofocus. You can set the focusing mode in Camera-Image.
Aperture Click to increase the aperture, to decrease the aperture, and for an adaptive aperture. You can set the aperture mode in Camera-Image.
Backlight Click to to turn on the IR lamp and turn off the IR lamp.
Note: This option is available only to IR cameras.
Wiper The wiper is used to clean the glass panel. Clicking will enable the wiper and clicking again will disable the wiper. If you do not manually disable the wiper, the wiper automatically stops working in ten minutes after being enabled.
Preset You can save, delete, and load presets. For details on how to configure a preset, see Camera-PTZ.
