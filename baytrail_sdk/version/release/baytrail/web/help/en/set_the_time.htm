On the Time tab page, you can configure time settings for the camera.

To manually set the time for the camera:

1. Click Manual Time Correction.

2. In the displayed dialog box, specify Time or select the Synchronize time with the PC check box.

3. Click Confirm.

To use Auto Time Correction:

1. Select the Auto Time Correction check box.

2. Select a time correction mode.

If you select Access Protocol, time correction will be performed using the VSIP, GB28181, Onvif, and SDK.

If you select NTP Clock Synchronization, time correction will be performed using the NTP. Additionally, you need to specify Server Address, NTP Port Number, and Correction Interval.

To configure the Daylight Saving Time (DST):

1. Select the Enable DST check box.

2. Specify Start Time and End Time.

3. Select a duration option from the Advance Clocks By drop-down list.

4. Click Save.
