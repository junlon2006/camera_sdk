To configure the network settings for the camera:

1. Select a mode.

2. Specify parameters displayed.

3. Configure DNS settings.

If you set Mode to Static, you cannot select the Automatically Obtain DNS Server Address check box.

The value range of the MTU parameter is from 500 to 1500. The default value is 1500.

Port                
HTTP Port The default value is 80.
HTTPS Port The default value is 5544. When the default value is used, the camera can be reached by https://X (X indicates the IP address of the camera). For example, https://198.12.12.12. When the default value is not used, the camera can be reached by https://X:Y (Y indicates the value of this parameter). For example, https://198.12.12.12:5000.
RTSP Port The default value is 554. When the RTSP is applied, the camera can be reached by rtsp://X/id=Z. Z can be set to 0 (main stream) or 1 (secondary stream).
