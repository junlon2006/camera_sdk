On the OSD tab page, you can enter the text that is to be displayed on the view window.
To enter the text:
1. Select one or multiple options under Contents to Be Displayed. Time: The Time option is selected by default.
Label: The Label option is selected by default. The default label is IPCamera.
OSDX: A maximum of six texts can be displayed on the view window.
2. Configure the format, font, and margin. Time Format: The default format is YYYY-MM-DD.
Type: Font type
Margin: Margin on the view window
3. Edit texts. Double-click a text box to edit the text.
Note: A maximum of 32 characters can be entered in a text box. You can enter up to 22 characters for the device label.
4. Change text and label locations. You can drag a text or label box to change its location.
Note: All the text boxes are accommodated in the large blue square. You may need to move the blue square before you can move a text box.
5. Click Save. The Save button is located in the top left corner.
Note:
1. One digit, letter, or punctuation is regarded as one character and one Chinese character is regarded as two characters.
2. You can click Load Default Scheme or Load Scheme to change the layout of texts and the device label.
