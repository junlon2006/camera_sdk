On the User Safety tab page, you can add, edit, and delete users.
Anonymous Access
When the Anonymous Access check box is selected, anonymous uses can access the camera.
Note: Anonymous users can only view the live video from the camera.
Add User

To add a user:

1. Click Add.

2. In the displayed Add User dialog box, specify parameters displayed.

3. Click Confirm.

Delete User

To delete a user:

1. Select the user from the user list.

2. Click Delete.

Edit User

To edit a user:

1. Select the user from the user list.

2. In the displayed Edit User dialog box, specify parameters displayed.

3. Click Confirm.

User Authorization
Administrator All options are available to an administrator.
Operator By default, the Live View, Playback/Snapshot, and PTZ Control options are available to an operator.
Browser A browser can only view the live video from the camera.
Note: Changes to the values of specific parameters apply only after the camera is restarted. In such a case, the user must be allowed to configure system settings and parameters and to restart the camera.
RTSP Authorization                
By default, Type is set to none.
IP Filter                              
On this page, you can create, edit, and delete a white list or black list.
To create a white list:

1. Set IP Filter to White List.

Note: By default, IP Filter is set to Disable.

2. Click Add.

3. Enter the target IP address.

4. Click Confirm and then Save.

Note: You cannot proceed with the configuration if you set IP Filter to White List and no IP addresses are added to the white list.
