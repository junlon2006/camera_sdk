To configure snapshot settings:

1. Specify Picture Format, Resolution, and Quality.

2. (Optional) Select Scheduled Capturing.

The Number of Snapshots Taken by Scheduling Task parameter indicates the number of snapshots that can be taken during a scheduled capturing task.

The Number of Snapshots Taken upon Occurrence of Event parameter indicates the number of snapshots that can be taken when an event is generated.

3. Click Save.

Note: Before enabling the camera to take snapshots, install TransFlash cards (specific to dome cameras) or Secure Digital cards (specific to bullet or box cameras). If you do not install these cards, the camera cannot take snapshots.
