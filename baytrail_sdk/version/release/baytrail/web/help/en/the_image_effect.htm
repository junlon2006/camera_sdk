Image Adjustment
Brightness The default value is 50.
Contrast Contrast is the difference in luminance and color that makes an object distinguishable. The default value is 50.
Saturation Saturation is the colorfulness of a color relative to its own brightness. The default value is 50.
Sharpness In photography, the term "acutance" describes a subjective perception of sharpness that is related to the edge contrast of an image. Due to the nature of the human visual system, an image with higher acutance appears sharper even though an increase in acutance does not increase real resolution. The default value is 50.
Restore Clicking this button will set Brightness, Contrast, Saturation, Sharpness and to their default value 50.
Exposure
Gain Mode

The greater the gain, the brighter the image and the more the image noise. You are advised to set Gain Mode to Auto.

Note:

1. If you select Auto, you need to specify Gain Upper Limit.

2. If you select Manual, you need to specify Gain Level.
Aperture Mode Aperture mode
Shutter Mode

The lower the shutter level, the darker the image. You are advised to set Shutter Mode to Auto.

Note:

1. If you select Auto, you need to specify Shutter Lower Limit.

2. If you select Manual, you need to specify Shutter Level.
Anti-Flicker You can select 50 or 60 Hz to prevent lights on images from flickering.

White Balance
White Balance - White balance is used to adjust the colors of a scene to accurately represent them. You are advised to select an auto mode.
Day/Night Switching
Day/Night Switching- Switching modes are described as follows.
Day In this mode, colors are retained.
Note: Infrared radiation will not be captured by the camera in this mode.
Night In this mode, images are in black and white.
Note: Infrared radiation will be captured by the camera in this mode.
Auto The Switching Time parameter indicates the time taken to switch from Day to Night or from Night to Day
Scheduled Day/Night At a scheduled time, the Night mode is switched to the Day mode.
Note: During the scheduled period, the Day mode takes effect.
Alarm Trigger Switching In this mode, an alarm can trigger a switching from Day to Night or from Night to Day.

Rotate and Echo
Corridor Mode If you select Left, images will be rotated counterclockwise.
If you select Right, images will be rotated clockwise.
Mirroring Mode When the camera is installed upside down, you can use this mode to rotate images.
Parameter values are described as follows:
Left and Right: Images are reversed.
Up and Down: Images are moved upside down.
Center: Images are rotated 180°.
Connect to Monitor Parameter values are Disable, PAL, and NTSC. PAL stands for Phase Alternating Line and NTSC stands for National Television System Committee. PAL and NTSC are the color encoding system.
