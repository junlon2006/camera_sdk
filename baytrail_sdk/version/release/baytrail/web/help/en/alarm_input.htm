Alarm Input                               Return to Contents

To configure an alarm input:

1. Specify parameters displayed.

Note: You can enter up to 32 characters for the Alarm Name parameter.

2. (Optional) Configure common linkage settings.

3. (Optional) Configure other linkage settings.

4. (Optional) Select the All check box to apply the settings of the alarm input to the other alarm inputs.

5. Click Save.

Note: The numbers of alarm inputs and outputs vary according to the camera type.
The following table provides the description for each action:
Report to Management System Send alarm messages to the camera management system.
Text Overlay Display the duration the alarm persists.
Play Alarm Sound Play the alarm sound.
Recording Start a recording.
Snapshot Take snapshots.
Alarm Output Evoke alarm outputs.
PTZ Linkage Go to a preset position. To use this option, select the PTZ Linkage check box and then a preset ID.
Note: This option applies only to PTZ cameras.
