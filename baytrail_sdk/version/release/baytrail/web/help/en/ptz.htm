Go to Last Spot upon Power-on

When this option is selected, the camera goes to a spot when powered off and then on.

To determine the spot, select an option from the When powered on drop-down list.

Patrol                
On the Patrol tab page, you can enable the Pan Patrol, Tilt Patrol, Load Preset, Manual Patrol, Frame Patrol, Random Patrol, Panoramic Patrol, and Track Patrol functions.
Before enabling these functions, select the Enable check box.
All these functions are listed in the Mode drop-down list.
Pan Patrol The pan patrol range is 0 ° - 180°. The Resumed In parameter specifies the time in which a patrol is resumed after being interrupted. The default value of this parameter is 1 minute.
Tilt Patrol The tilt patrol range is 0 ° - 180°. The Resumed In parameter specifies the time in which a patrol is resumed after being interrupted. The default value of this parameter is 1 minute.
Load Preset The camera goes to a preset when it does not receive any PTZ command for a period specified by theResumed In parameter. The default value for this parameter is 3 minutes.
Path Patrol The camera patrols paths when it does not receive any PTZ command for a period specified by theResumed In parameter. The default value for this parameter is 1 minute.
Frame Patrol The camera pans 90° upon receiving every other key frame. The Resumed In parameter specifies the time in which a patrol is resumed after being interrupted. The default value of this parameter is 1 minute.
Random Patrol The camera randomly pans. The Resumed In parameter specifies the time in which a patrol is resumed after being interrupted. The default value of this parameter is 1 minute.
Panoramic Patrol The camera pans from 0 ° to 360° and tilts from 0 ° to 180°. The Resumed In parameter specifies the time in which a patrol is resumed after being interrupted. The default value of this parameter is 1 minute.
Track Patrol The camera patrols history tracks. The Resumed In parameter specifies the time in which a patrol is resumed after being interrupted. The default value of this parameter is 1 minute.
Preset               

To configure a preset:

1. Select a position using direction arrows.

2. Configure the focus and aperture.

3. Select an ID.

4. Name the preset.

5. Click Apply.

6. (Optional) Click Load to load the preset.

Border                
Borders are used to limit the viewing and patrol range of the camera.
To configure borders:

1. Move the camera to a position using the left direction arrows.

2. Click Left Border.

3. Move the camera to a position using the right direction arrows.

4. Click Right Border.

To clear all preset borders, click Clear Borders.

When Enable Manual Restrict Patrol Area is selected, the preset borders take effect when you move the camera to look around the area under surveillance.

When Enable Auto Restrict Patrol Area is selected, the presets borders take effect during a patrol.

Path Patrol                 
All the 256 presets can form 8 paths. When a path is selected, the camera can patrol 32 presets. Presets 1 through 32 are assigned to path 1, presets 33 through 64 to path 2, and likewise the rest presets.
To add a preset to a path:

1. Select a path from the Patrol Path drop-down list.

2. Click Add.

3. Specify parameters displayed in the Add dialog box.

4. Click Confirm.

Track Patrol                 
During a track patrol, the camera patrols all the positions that the camera goes to during a specific period of time. A maximum of four tracks can be created.
To create a track:

1. Select a track from the Patrol Track drop-down list.

2. Click Start Recording.

3. Move the camera using PTZ control buttons.

4. Click Stop Recording. Note that each recording lasts for up to 10 minutes.

To preview a track, click Preview. To stop a preview, click Stop.

To remove all the recorded PTZ controls from a track, click Clear Records.

Scheduled Task                 

To schedule a task:

1. Select the Enable Scheduled Task check box.

2. Specify Resumed After. A scheduled task (for example, pan patrol) is occasionally interrupted by PTZ control operations. This parameter specifies the time after which the task is resumed.

3. Click Edit.

4. Select a day.

5. Select one or multiple durations.

6. Select a task from the Option drop-down list for each duration. For specific tasks, you need to specify ID.

7. (Optional) Copy the settings to other days.

8. Click Confirm.

Position               
When you click Zero Correction, the camera goes to the mechanical zero point.
To change the position of a camera:

1. Specify Horizontal Coordinate.

2. Click Horizontal Position.

3. Specify Vertical Coordinate.

4. Click Vertical Position.

Alternatively, specify Horizontal Coordinate and Vertical Coordinate, and then click Position.
