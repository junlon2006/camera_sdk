On the Live View tab page, you can view the live video from the camera. In the top right corner of the view window, you can select a stream to view.

Double-clicking the view window will enter the full screen mode. Double-clicking the view window again or pressing Esc will return.

Note: The number of streams varies according the camera type.
Main Stream The main stream is usually recorded. To configure the main stream, see Encoding.
Secondary Stream

When bandwidth resources are insufficient, you can view the secondary stream with a lower encoding rate and a lower resolution. To configure the secondary stream, see Encoding.

Third Stream

You can view the third stream with a lower encoding rate and a lower resolution on your mobile phone. To configure the third stream, see Encoding.

Fourth Stream The fourth stream is reserved for the Intelligent Analysis feature. To configure the fourth stream, see Encoding.
