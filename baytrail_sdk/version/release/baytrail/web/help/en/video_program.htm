On the Record tab page, you can schedule a recording.
To schedule a recording:

1. Specify parameters under Basic.

Note: When specifying Pre-Recording For and Prolong Recording For, you need to consider the encoding rate (Camera-Video-Encoding ) of the stream in question.

2. Click the Enable check box.

3. Click Edit to specify a start time and an end time.

4. Click Confirm and then Save.
