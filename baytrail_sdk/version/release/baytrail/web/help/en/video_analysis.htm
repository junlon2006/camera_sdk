Motion Detection                                Return to Contents
The motion detection feature can trigger an alarm when a suspicious motion is detected in a preset area. By default, the alarm is cleared in 5s.
To use this feature:

1. Select the Enable check box.

2. Click Edit next to Field.

3. Select a field using your mouse. Note that you can select a maximum of four fields.

4. Specify Sensitivity.

5. Click Edit next to Time to specify the time when this feature takes effect.

6. Configure linkage settings.

7. Click Save.

Mask Detection                 
The mask detection feature can trigger an alarm when a preset area is masked.
For details on how to use this feature, see the preceding steps for reference.
