To register with Kedacom Video Management System (VMS) using VSIP:

1. Select the Enable check box.

2. Specify parameters displayed.

3. (Optional) Select the Send NAT Probe Packets check box when the camera is located in an NAT-enabled network.

4. (Optional) Select the Auto Networking check box to enable the Auto Networking function.

Note: You can use VSIP to register only with Kedacom VMS or Kedacom-SDK-integrated management systems. SDK stands for software development kit.
Onvif                

To register with an ONVIF-compatible management system, select the Enable check box.

When the Enable check box is selected, a URL is generated. The camera can then be reached through the URL. The default authorization mode is WS-Username token.

GB28181                

To register with a management system complying with a guo biao (GB) standard:

1. Select a platform.

2. Select the Enable check box.

3. Specify parameters displayed.

4. Click Save.
