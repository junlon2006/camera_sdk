Limited Encoding                
Limited Encoding allows the camera to encode and display the video of a specific field.
To use Limited Encoding:

1. Select the Enable check box.

2. Click Edit.

3. Select a field using your mouse.

4. Click Done and then Save.

Note: You cannot select a very small field.
ROI                

When a region of interest (ROI) is created, the image quality in the ROI is high but that outside the ROI is low. In such a case, you are advised to decrease the encoding rate.

To create an ROI:

1. Select the Enable check box.

2. Click Edit.

3. Select a field using your mouse.

4. Select an encoding rate level.

5. Click Done and then Save.

Note: If you change the resolution and aspect ratio of the camera, an ROI no longer takes effect.
Privacy Mask                
The privacy mask feature prevents a camera from monitoring sensitive objects or areas in a scene. When privacy masks are created in a scene, you cannot track objects if the objects move to a masked area. Therefore, you are advised to disable this function when you are tracking moving objects.
To create a privacy mask:

1. Select the Enable check box.

2. Click Edit.

3. Select a field using your mouse.

4. Select a mask color.

5. Click Done and then Save.

Note: You can select a maximum of four fields.
