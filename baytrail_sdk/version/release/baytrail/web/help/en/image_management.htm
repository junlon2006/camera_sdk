On the Snapshot tab page, you can search and download snapshots that are taken manually or taken upon alarm generation.

To search snapshots those are taken upon alarm generation:

1. Select Snapshots Taken upon Alarm Generation under Search Criteria.

2. Select a time option.

3. Click Search.

To download a snapshot, select the snapshot and click Download. By default, the snapshot is saved in the directory C:\IPCWeb\Pic\PuGrab. To change the save path, choose Settings > Local > Snapshot > Camera: snapshot save path.
