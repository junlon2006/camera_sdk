录像                               返回目录
启用录像计划后，摄像机可在设置的时间段内自动进行录像。
设置方法：

1.选择录像类型：即选择具体录像的码流，具体码流的配置参考摄像机-视频-视频编码界面；

说明：具体选择录制哪路码流，要充分计算所选码流的码率，这与录像时间直接相关。

2.根据接入协议的类型选择录像打包格式；

3.选择预录方式；

4.设置录像延时时间，即在所设置的录像时间结束后继续录像的时间；

5.勾选启用录像计划，点击编辑按钮，在弹出的界面中勾选时间段，并设置计划录像的开始时间和结束时间。点击复制按钮，可以将设置的录像计划时间复制到其他某一天或一整星期。设置好后点击确认按钮；

6.保存设置。
