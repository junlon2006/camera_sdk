在OSD界面可设置画面中显示的字幕内容。
配置方法：
1.选择显示内容 时间：勾选则默认在图像右上角显示时间，不选不显示。默认：勾选。
台标：勾选则默认在图像左上角显示IPCamera，不选不显示。默认：勾选。
OSD：可勾选OSD1~OSD6，最多设置6处字幕，不选不显示。默认：不勾选。
2.编辑字体格式 时间格式：可选择时间字幕的显示方式，支持：“年-月-日”、“日-月-年”、“月-日-年”三种显示方式，默认：“年-月-日”。
多行时间：勾选后则日期和时间分行显示，默认：不勾选。
字体：支持黑体、宋体、Arial 三种字体格式，默认：黑体。字体大小支持选择，并支持自定义可满足不同分辨率下设置不同的字体大小。
边距：调节字幕和边框的距离。
3.编辑字幕内容 在画面中双击需设置的OSD文本框，弹出字幕设置窗口，输入文字，点击确定。
说明：每个OSD最多设置64个字符，台标最多设置64个字符。
4.编辑字幕坐标 选中字幕区域，按住鼠标拖动，即可改变位置。
说明：每个OSD只能在蓝框中拖动，如果需要移出蓝框位置需先移动整个蓝框位置。
5.保存设置 点击左上角保存按钮。
说明：
1.一个汉字占用一个字符，一个数字、英文、标点等占用一个字符。
2.也可点击载入默认方案或载入国标方案，载入后也可按照上面的步骤设置显示内容、编辑位置、编辑字幕信息。
