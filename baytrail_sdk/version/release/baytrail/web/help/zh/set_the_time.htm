在时间设置界面，可以设置时区、校时方式和夏令时。
设备时区-显示当前所在时区。
设备时间--显示当前时间。
设备时区-显示当前所在时区。
手动校时-即自行输入时间或同本地电脑同步。
设置方法：

1.点击手动设置按钮；

2.在弹出的手动设置时间界面中选择时区并设置时间。若勾选与计算机时间同步，则设置为时间与计算机时间同步；

3.点击确定后生效；

自动校时-即摄像机通过接入协议或NTP服务自动进行时间校准。
设置方法：

1.勾选开启自动校时；

2.若选择接入协议校时，则系统根据接入协议自动校准时间；若选择NTP校时，则需要填写NTP时间同步服务器地址和端口，并设置校时时间间隔。

夏令时-摄像机支持启用夏令时，夏令时也是一种计时方式。：
设置方法：

1.勾选启用夏令时；

2.设置开始时间和结束时间，设置方式为在下拉框中选择开始或结束时间为X月的第X个星期X的某一时间，如设置开始时间为4月的第二个星期日的02时，结束时间为9月的第一个星期日的02时；

3.在下拉框中选择偏移时间，即实行夏令时的偏移时间，可选择30分钟/60分钟/90分钟/120分钟，默认为30分钟。
