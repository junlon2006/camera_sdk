录像回放                               返回目录
在菜单栏中点击录像回放进入录像回放界面。通过录像回放界面，可以进行搜索、查看、下载录像等操作。
回放录像步骤：

1.选择录像时间，在录像月视图选择需要查看录像的日期。如果月视图中的某个日期有底色，代表该天有监控录像存储；

2.在录像类型中勾选需要查看录像的类型，可全选或者告警录像、计划录像、手动录像自由组合；

3.点击查找按钮，在时间轴上可以选取选择当天录像中具体时间段的录像信息；

4.点击界面下方的播放按钮即可播放录像；

5.鼠标箭头放在录像时间轴可显示录像时间信息，单击鼠标左键或者按住所标左键再左右移动时间轴可进行录像跳播。

—点住鼠标左键不放，在时间轴上左右拖动，可滑动时间轴。

—鼠标在时间轴上移动，时间轴上方可现实时间位置信息。

—鼠标单击时间轴上的某个点，在时间轴的中间位置显示该点的时间位置信息。

在回放录像过程中，可以通过录像回放下方的工具栏对回放的录像进行裁剪、快进、下载等操作。
/ 播放/暂停，点击按钮播放录像，再次点击暂停播放。
停止，点击该按钮，停止录像播放。
减慢播放速度，点击减慢播放速度。点击一次减慢至1/2x，再次点击减慢至1/4x，最低减慢至1/4x。
加快播放速度，点击加快播放速度。点击一次播放速度加快一倍，最高调节到8x。
前一片段，点击播放上一帧图像，可连续点击。
后一片段，点击播放下一帧图像，可连续点击。
声音，点击打开声音，再次点击关闭声音。拖动滑块可调节音量。
EPTZ，电子云台功能。单击按钮，点住鼠标左键不放，向右下方框选出某一区域，可以扩大该区域的像素，并使该区域铺满画面。点住鼠标左键不放，向左上方框选区域，则恢复画面。
抓拍，点击即可抓拍当前回放画面。回放抓图保存路径可在配置--本地配置界面的“回放抓图保存路径”中设置，默认路径为：C:\IPCWeb\Pic\PlayBack。
剪辑，点击即可开始对当前浏览录像进行剪辑，再次点击则停止剪辑。回放剪辑录像的保存路径可在配置--本地配置界面的“回放剪辑路径”中设置，默认路径为：C:\IPCWeb\Video\PlayBackCut。
下载，点击该按钮，即可下载录像并保存在PC机上。在弹出的录像文件下载页面中，可以查看录像下载进度。下载录像的保存路径可在配置--本地配置界面的“回放下载保存路径”中设置，默认路径为：C:\IPCWeb\RecPlayback。
/ 放大/缩小时间轴，即调整时间轴单位刻度值。点击即可放大或缩小时间轴。时间轴单位刻度值等级为5分钟/10分钟/30分钟/1小时/2小时。放大或缩小时间轴不影响当前播放时间点。
全屏，点击该按钮，显示全屏画面。在全屏画面中双击鼠标左键或者按Esc键退出全屏。
