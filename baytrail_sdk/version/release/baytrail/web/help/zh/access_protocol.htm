客户端支持KEDA VSIP，配置步骤：

1.勾选启用注册平台；

2.在注册平台地址下拉框中选择IP地址或域名；

3.在上一步若选择IP地址则在注册平台IPV4地址中输入注册平台地址，若在上一步选择域名，则在注册平台域名中输入平台域名；

4.填写注册平台端口信息，默认为5510，一般注册端口后不需要更改；

5.填写入网设备UUID和密码；

6.当摄像机设备在NAT内时，勾选发送探测包选项；

7.勾选支持自动组网，则开启自动组网功能。

说明：VISP协议只支持注册科达或者通过科达SDK集成的后端管理平台。
Onvif                               
勾选开启Onvif，可入网到支持Onvif协议的后端管理平台，默认端口为80。开启后，设备自动生成服务器地址（URL）。
GB28181                               
GB28181配置步骤：

1.勾选开启GB28181；

2.填写本地端口信息（默认为5060）、入网ID、设备名称、平台ID、平台IPv4地址、平台端口信息、注册用户名和密码、续租时间、心跳间隔等信息。
