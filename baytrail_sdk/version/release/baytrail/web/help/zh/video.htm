视频编码                               返回目录
在视频编码界面可设置编码参数。
多码流 可选择码流数量，更改后需要重启才能生效。默认：双码流。
说明：支持的码流数量和具体产品型号相关，如有疑问请核对产品具体参数。
码流类型 当选择双码流及以上码流数量时，可通过码流类型选择需要设置的码流，选中后即可设置选中码流的分辨率、码率的参数。
分辨率 在下拉列表中选择图像显示的分辨率。
码率类型 可选择定码率或变码率，对视频码流大小进行控制。默认：定码率。
视频帧率 设置编码每秒帧数。
说明：在三码流模式下，主码流视频帧率会略微下降。
码率上限 设置码率上限值。
编码格式 选择视频编码格式，支持H.264和MJPEG，默认;H.264。
编码复杂度 选择编码复杂度。
说明：编码复杂度越高，码流压缩就越大。这样能适当缓解带宽限制，但同时会占用更多设备的CPU资源。建议采用默认等级。
1帧间隔 最大关键帧间隔，1~250可设置，建议采用默认值75帧。
编码裁剪                               
创建自定义编码区域后，系统只对该区域进行编码和显示。
设置方法：

1.勾选启用编码裁剪；

2.点击开始绘制按钮，在左侧图像区域框选需要裁剪的区域；

3.点击保存。也可点击清除绘制可重新选定区域。

说明：要增加绘制区域有最小面积要求。
ROI                               
将码率资源集中在选定的感兴趣区域，提升框选区域的图像质量，但同时会牺牲选定区域外的图像质量。实现保证关键区域图像质量的前提下，可同时降低码率。
设置方法：

1.勾选启用敏感区域编码；

2.点击开始绘制按钮，在左侧图像区域框选需要裁剪的区域，并在等级下拉框中选择编码等级；

3.点击保存。也可点击清楚绘制可重新选定区域。

说明：启用敏感区域编码后，如果对图像采集分辨率或采集高宽比（标屏/宽屏）进行更改，设备将自动退出敏感区域编码。
视频遮蔽                               
对图像敏感区域进行遮蔽，遮挡不希望被看到的区域。
设置方法：

1.勾选开启视频遮蔽；

2.点击开始绘制按钮，左侧视频图像被划分成16列12行的小方格，点击选中画面中的小方格确认遮蔽区域；

3.可在遮蔽颜色下拉框中选择遮蔽颜色；

4.点击保存。也可点击清空绘制可重新选定区域。

说明：摄像机最多同时支持4个区域的图像遮蔽。
