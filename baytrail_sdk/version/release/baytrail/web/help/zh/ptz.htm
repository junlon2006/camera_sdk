云台版本 显示云台版本。
扫描速度等级 设置巡航扫描时的速度等级（1~40），数值越大，扫描速度越快。
景深比例 勾选开启景深比例。在不同视野下，云台转动保持相同的线速度。当视野放大时，云台减小线速度；视野缩小时，云台增大线速度。
断电恢复 1.勾选开启断电恢复；
2.选择恢复方式:
上电加载预置位：即在断电重启后加载到设置的预置位。选择上电加载预置位后，填写需加载的预置位编号。
断电记忆：云台在同一个位置停止时间达到设置时间，自动记录云台当前位置。设备重启后，云台会自动载入这个位置。默认设置时间为60s。
说明：预置位需提前设置好才能调用。
云台恢复出厂 所有已设的云台设置将被重置为出厂设置。
云台升级 云台升级是对球机固件进行升级，非特殊情况下，无需升级。云台升级需获取本地升级文件<*.bin>，在升级过程中，请耐心等待，并不能做其他操作。
守望设置                               
守望功能指球机在设定的时间内没有接收到控制指令就能运行所设定的动作。可设定动作包括：水平扫描、垂直扫描、载入预置位、路径巡航、帧扫描、随机扫描、全景扫描和花样扫描。
使能-勾选开启守望任务。
守望模式-在下拉框中选择守望模式。
水平扫描 摄像机自动从水平位置0°到360°水平巡视。时间可设置，默认1分钟。
垂直扫描 在摄像头允许的角度范围中自动垂直（0°至180°）扫描，时间可设置，默认1分钟。
载入预置位 停止对摄像头操作的一段时间后，摄像头将自动回归到用户自己设定的预置位。时间可设置，默认3分钟。预置位设置可参考预置位设置部分。
路径巡航 停止对摄像头操作的一段时间后，摄像机自动启动路径巡航功能。时间可设置，默认1分钟。
帧扫描 每隔一个关键帧间隔时间，摄像机自动水平转动90°，时间可设置，默认1分钟。
随机扫描 摄像机自动在水平方向上左右随机巡视，时间可设置，默认1分钟。
全景扫描 摄像机进行水平方向0°~360°以及垂直位置0°~180°全面巡视，时间可设置，默认1分钟。
花样扫描 摄像机按已记录的时间段内云台运动轨迹进行巡视，时间可设置，默认1分钟。
预置位                               
预置位是一组包含云台位置、镜头焦距的信息。由用户设置保存，并可随时调用，更改删除。
调节云台到指定位置，并根据需要调整视野、光圈、焦距，在预置点编号下拉框中选择编号，在预置点名称中输入预置点名称，点击设置按钮，即完成预置位的设置。可依次设置不同的摄像机位置。点击调用按钮，即可调用相应编号的预置位。点击删除按钮，则删除相应编号的预置位。
限位                               
限位分为键控限位和扫描限位。启用限位功能后，可以对云台扫描的边界进行设置。
设置方法：

勾选启用键控限位或扫描限位功能，调节云台到合适位置，并设置当前位置为左限位，再次调节云台位置，设置为右限位。设置完成后，立即生效。若要清除限位，点击限位清除按钮即可。

设置后，如果开启键控限位功能，则手动控制云台的转动只能在左限位和右限位之间进行进行转动；如果开启扫描限位功能，则云台只能在左限位和右限位之间进行水平扫描。

巡航扫描                               
巡航扫描是指对某一路径上所有有效设置的预置位进行巡航。开启该功能后，点击任意云台控制按钮即可停止该功能。目前，路径1包括1~32号预置位，路径2包括第33~64号预置位，以此类推。
操作方法：

1.在巡航路径下拉框中选择巡航路径；

2.点击添加按钮，在弹出的界面中添加预置点并设置停留时间。每条路径最多可添加32个预置点；

3.点击保存；

4.根据操作需要点击相应按钮进行调用、删除、停止巡航扫描的操作。

花样扫描                               
花样扫描是指摄像机按已记录的时间段内云台运动轨迹进行巡视。一共可设置4条花样扫描路径。
设置方法：

在扫描路径下拉框中选择花样扫描路径，点击开始记录按钮，搭配调整云台控制，记录云台转动、步长、视野拉伸操作，点击停止记录按钮，停止记录，每条路径最长记录十分钟。

点击预览按钮，浏览所记录的云台操作。点击停止按钮，停止预览。

选择路径，点击记录清除按钮，即可删除该路径下所记录的云台操作。

定时任务                               
启用定时任务，设置定时任务时间及操作任务，在任务时间内，摄像机执行其操作任务。
设置方法：

1.勾选启用定时任务；

2.输入任务恢复时间（5~720s），例如5秒，在任务时间内进行其他云台操作，5秒后，摄像机恢复其定时操作任务；

3.点击编辑按钮，在弹出的界面中勾选时间段，选择开始时间和结束时间，不同时间段编号项所设置的时间段不可重叠；

4.在守望模式下拉框中选择需执行的定时任务；

5.设置编号，如果任务有多条扫描路径或多个预置位，编号可设置（如花样扫描有多条扫描路径，则任务参数可设置为扫描路径编号值；如加载预置位，可设置预置位编号值），反之，则无法设置；

6.复制，点击复制按钮，可以将设置的任务时间复制到其他某一天或一整星期。

定位                               
零位矫正：当进行载入预置位等云台操作时，各位置都出现错位，此时必须进行零位校正。零位校正后，摄像机的零位将自动被矫正到机械零位。点击零位矫正按钮，即可进行零位矫正。
定位方法：

调节云台到指定位置，即可显示当前位置的坐标信息。根据当前位置的坐标信息，在水平坐标和垂直坐标栏中填写目标位置坐标信息。点击水平定位按钮，即可在水平方向转动云台至水平坐标位置。点击垂直定位按钮，即可在垂直方向转动云台至垂直坐标位置。点击精准定位按钮，即可转动云台至水平坐标和垂直坐标目标位置。
