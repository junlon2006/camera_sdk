登陆客户端后默认进入实时浏览界面。通过实时浏览界面可以看到摄像机的当前拍摄画面。
在实时浏览界面，双击画面即可全屏显示，再次双击或者按Esc键退出全屏。
在实时浏览界面可以对视频浏览的码流进行选择。对同一视频源，同时编码几路不同分辨率的视频，用户可以根据带宽情况在浏览窗口的右上方选择。
说明：请根据具体设备性能确定多码流路数。
主码流 调取视频编码配置下的主码流视频，一般用于存储或监控。
辅码流 调取视频编码配置下的辅码流1视频，可适当降低码率和分辨率，在宽带有限的情况下实现远程监控。
第三码流 调取视频编码配置下的辅码流2视频，可适当降低码率和分辨率，通过无线网络实现手机等终端设备的远程监控。
第四码流 调取视频编码配置下的辅码流3视频，专门提供给智能分析应用。
在浏览窗口的上方可设置视频窗口显示比例。
点击该按钮，图像按照4:3标屏显示
点击该按钮，图像按照16:9宽屏显示
点击该按钮，显示视频码流的实际尺寸
点击该按钮，视频窗口显示自动适应系统分辨率
通过浏览窗口下方的工具栏按钮可以对当前浏览画面进行播放控制。
/ 开始/暂停，点击该按钮，播放视频浏览画面，再次点击，暂停播放。
停止，点击该按钮，停止视频浏览。
音量，本地解码音量，拖动滑块可调节音量大小。
语音呼叫前端，点击开始对讲，再次点击停止对讲。
抓拍，点击即可抓拍当前画面。抓拍图片保存到本地配置中预览抓图保存路径，默认保存路径为：C:\IPCWeb\Pic\Preview。
开始录像/停止录像，点击按钮开始录像，再次点击停止录像。录像保存在本地配置中录像文件保存路径，默认保存路径为：C:\IPCWeb\Rec。
EPTZ，电子云台功能。单击按钮，点住鼠标左键不放，向右下方框选出某一区域，可以扩大该区域的像素，并使该区域铺满画面。点住鼠标左键不放，向左上方框选区域，则恢复画面。
PTZ点击缩放，单击按钮，点住鼠标左键不放，向右下方框选出某一区域，该区域将被放大并在画面居中。点住鼠标左键不放，向左上方框选区域，就可以取消放大。在画面中双击某个点，使该处居中显示。
说明：只有带PTZ功能的摄像机支持。
状态，点击显示帧率和码率，再次点击隐藏帧率和码率。
全屏，点击该按钮，显示全屏画面，在全屏画面中双击鼠标左键或按Esc键退出全屏。
