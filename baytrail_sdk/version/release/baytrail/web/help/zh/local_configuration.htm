本地配置                               返回目录
在客户端的本地配置界面，可以对摄像机的播放参数以及抓拍图片、录像的保存路径进行设置。
播放参数
协议类型 选择码流的输出协议，可选择UDP或TCP传输，默认：UDP。
播放性能 选择实时性好、均衡或流畅性好三种播放等级。默认：实时性好。
图像降噪 图像降噪为解码端降噪，勾选开启降噪只改变当前用户的浏览效果。勾选降噪后可通过下面进度条选择降噪等级，降噪等级1-4可调，等级越大降噪越明显。
说明：图像降噪等级太高会导致运动物体拖尾，正常情况下无需开启。
垂直同步 当图像出现撕裂时，开启垂直同步可以改善。
说明：开启会增加CPU能耗，正常情况下无需开启。
录像
录像文件打包大小 选择实时浏览界面点击录像按钮后，生成在PC机的每个录像文件的大小。默认：512M。
浏览录像保存路径 设置在实时浏览界面点击录像按钮后，保存在PC机的路径。默认路径为：C:\IPCWeb\Rec，可点击浏览按钮自定义保存路径。
回放剪辑保存路径 设置在录像回放界面点击剪辑按钮后，保存在PC机的路径。默认路径为：C:\IPCWeb\Video\PlayBackCut，可点击浏览按钮自定义保存路径。
回放下载保存路径 设置在录像回放界面点击下载按钮后，保存在PC机的路径。默认路径为：C:\IPCWeb\RecPlayback，可点击浏览按钮自定义保存路径。
抓拍
浏览抓拍保存路径 设置在实时浏览界面点击抓拍按钮后，保存在电脑中的路径。默认路径为：C:\IPCWeb\Pic\Preview，可点击浏览按钮自定义保存路径。
前端抓拍保存路径 设置在图片管理界面点击下载图片后在PC机的保存路径。默认路径为C:\IPCWeb\Pic\PuGrab，可点击浏览按钮自定义保存路径。
回放抓拍保存路径 设置在录像回放界面点击抓拍按钮后，将录像回放时的抓图保存到电脑中的路径。 默认路径为：C:\IPCWeb\Pic\PlayBack，可点击浏览按钮自定义保存路径。
下载插件
下载插件 点击下载插件按钮，下载视频插件。首次登陆客户端，需要下载并安装插件才能正常进行视频浏览。
