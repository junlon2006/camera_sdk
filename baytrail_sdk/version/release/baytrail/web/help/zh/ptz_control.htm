PTZ控制                               返回目录
PTZ控制界面可实现对云台的转动，镜头视野、聚焦和光圈的调节，预置位的设置，以及灯光和雨刷等的控制。支持镜头控制的摄像机设备或在连接好摄像机与网络适配器的串口控制线后，可以进行云台控制。
说明：出厂不带云台的设备，PTZ控制界面默认隐藏，需要时点击视频左侧隐藏按钮显示PTZ控制界面。
方向键 八个不同的方向键表示云台转动的不同方位，点击即可调整云台转动的方向。
云台复位 为云台复位按钮，点击使云台回到初始位置，并使视野恢复最大。
步长 步长为每次按动方向键，云台所转动的速度。拖动云台方向键右侧的滑块 ，即可调整步长。步长越大，点击后摄像机的移动速度越快。带电动变焦镜头的摄像机，步长也可控制镜头Zoom和Focus步进。
视野 视野指的是通过IPC监控到的空间范围。视野越大，看到的范围越大。点击可拉近视野，点击可拉远视野。
聚焦 调整聚焦的目的是使目标成像清晰。点击可缩小聚焦，点击可扩大聚焦，点击可实现自动聚焦。也可在摄像机-图像界面设置聚焦模式。
光圈 光圈控制镜头的进光量，大光圈进光量多，图像明亮。点击可调大光圈，点击可调小光圈，点击可实现光圈自动调节。也可在摄像机--图像界面设置光圈模式。
开灯/关灯 点击开启红外灯检测，将在光线条件变暗时开启红外灯进行辅助照明。点击强制关闭红外灯，这时即使光线变暗也不会开启红外补光灯。
说明：需要IPC设备支持该功能。
雨刷 雨刷有清洁摄像机玻璃罩的作用。点击打开雨刷，再次点击可关闭雨刷。开启雨刷后不关闭，雨刷工作十分钟后将自动停止。
预置位 在预置位列表中点击相应预置位序号，即可对相应预置位进行调用、设置、清除的操作。预置位的设置方法可参考摄像机-云台章节。
