图像调节
亮度 拖动滑块调节当前图像的亮度。如果将亮度调高，图像将更明亮。用户可根据需要适当调节，默认：50。
对比度 拖动滑块调节当前图像的对比度。如果将对比度调高，图像的明暗对比将更清晰。用户可根据需要适当调节，默认：50。
饱和度 拖动滑块调节当前图像饱和度。如果将饱和度调高，图像将更鲜艳。用户可根据需要适当调节，默认：50。
锐度 拖动滑块调节当前图像的锐度。如果将锐度调高，图像中物体的轮廓将更清晰。用户可根据需要适当调节，默认：50。
曝光
增益 当把增益调大时，会加强图像的亮度，但同时也会放大图像中的噪点。所以，推荐选择自动模式。
说明：
1.如果选择自动模式，可在下方拖动滑块设置增益上限值。这时增益在自动模式下就只能在0到设置的上限值内自动调节。
2.如果择手动模式，可在下方拖动快速设置增益等级。这时增益将始终保持设置的数值。
光圈模式 光圈模式可选择自动模式或手动模式。若选择手动模式，可在下方拖动滑块调节光圈大小和光圈灵敏度。
快门模式 快门功能指监控画面中像素点曝光的快门速度，快门越小图像越暗。可选择自动模式或手动模式，推荐选择自动模式。
说明：
1.如果选择自动模式，可在下方快门下限下拉框中设置快门下限值。这时快门在自动模式下就只能在设置的下限值到最小快门值之间自动调节。
2.如果选择手动模式，可在下方快门等级下拉框中设置快门等级。这时快门将始终保持设置的数值不变。
防闪烁 当在室内日光灯等照明环境下图像中出现上下浮动的横条纹时，可通过选择防闪烁的正确频率（50Hz或60Hz）解决。
聚焦
说明：并非所有型号支持此功能，请根据设备具体支持情况设定。
聚焦模式 可选择自动聚焦或手动模式。
最小聚焦距离 即镜头最短的聚焦距离，小于此距离，镜头将无法自动聚焦清楚。在下拉框中选择最小聚焦距离值。
说明：此功能针对球机产品，且选择自动聚焦模式的时候才生效。
白平衡
白平衡-在不同的光线照射下，图像会产生不同的偏色，调节该处，使图像还原到最逼真的状态。在白平衡下拉框中选择白平衡模式，建议使用自动模式。
日夜转换
日夜转换-在下拉框中选择日夜转换模式。
日模式 选择日模式图像始终保持彩色。
说明：此时不支持感红外。
夜模式 选择夜模式图像始终保持黑白。
说明：此时能感红外光。
自动模式 1.选择自动模式，将根据当前环境的光照度自动切换彩色和黑白。
2.可选择灵敏度（高、中、低，默认：中），控制日夜模式切换对环境光线变化的灵敏度。
3.拖动滑块调节转换时间，即根据环境光线变化由日模式转换到夜模式或由夜模式转换到日模式的时间。
定时模式 1.选择定时模式，将根据所设置的时间进行日夜转换。
2.点击编辑时间按钮，在弹出的编辑时间界面中设置开始时间和结束时间。
说明：选择定时模式并设置时间后，在所设置的时间段内为日模式，在时间段外为夜模式。
报警输入触发模式 即由报警输入触发日夜转换。选择报警输入触发模式后，在触发状态下拉框中选择白天或者黑夜。
图像增强
增强图像显示，当监控画面较为模式时，可尝试调整此参数。
图像中出现噪点时，可以调节降噪等级。
3D降噪模式-在下拉框中选择3D降噪模式。
3D降噪-拖动滑块调节3D降噪等级。
说明：3D降噪等级太高会导致运动物体拖尾，正常情况不需要改动。
动态调节-遇到逆光场景时，图像明暗对比特别强烈，可以通过动态调节优化图像效果。可选择：关闭、背光补偿（自动）、背光补偿（手动）、宽动态或者强光抑制。
关闭 关闭动态调节，默认动态调节是关闭的。
背光补偿（自动） 1.在动态调节下拉框中选择背光补偿（自动）；
2.拖动滑块调节背光补偿灵敏度，默认：50。
背光补偿（手动） 1.在动态调节下拉框中选择背光补偿（手动）；
2.拖动滑块调节背光补偿灵敏度，默认：50；
3.在背光补偿区域下拉框中选择补偿区域，默认：中心。
说明：若选择自定义，则点击图像下方的绘制区域按钮，拖动鼠标绘制背光补偿区域，然后点击设置即可。点击清除全部按钮，即可清除所绘制的区域。
宽动态 1.在动态调节下拉框中选择宽动态；
2.拖动滑块调节宽动态灵敏度，默认：50。
强光抑制 1.在动态调节下拉框中选择强光抑制；
2.拖动滑块调节强光抑制灵敏度，默认：50。
翻转与回显：
走廊模式 在狭窄的走廊、过道等区域可获得更多的有效视野。在下拉框中选择:
关闭：默认关闭，图像不翻转。
向左：图像向左翻转90°。
向右：图像向右翻转90°。
镜像模式 在摄像机倒装时，可通过此功能使得图像正立，无需重新安装摄像机。
关闭：默认关闭，图像不翻转。
左右：图像左右颠倒，但是上下还是相反的。
上下：图像上下颠倒，但是左右还是相反的。
中心：图像翻转180°。
本地回显 摄像机支持本地输出时可选择关闭、PAL或NTSC。
恢复默认：
恢复默认 点击恢复默认按钮，各项参数（除了翻转与回显）恢复默认。
