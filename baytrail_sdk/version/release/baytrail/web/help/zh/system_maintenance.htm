系统维护                               返回目录
在系统维护界面可以完成设备的重启、升级等维护类操作。
重启-点击重启按钮，即可重新启动设备。
发现-闪灯用于设备发现。在闪灯时间下拉框中选择闪灯时间，点击闪灯按钮。
说明：需要设备支持。
自动维护-自动维护用于需要设备定时重启的场合，一般为自动重启设备（非复位）。
设置方法：

1.勾选是否自动维护；

2.在维护周期下拉框中选择系统自动维护周期；

3.在维护时间选项中设置自动维护的时间；

4.点击保存。

升级-系统版本升级。选择获取本地升级文件<*.pkg>。升级过程中，请耐心等待，并请不要做其他操作。升级完成后，请重新登陆客户端。
简单恢复-简单恢复保留了基本网络参数和接入协议配置。
完全恢复-完全恢复即完全恢复参数到设备出厂设置。
配置导出-设置好摄像机模式后，可将其导出保存到本地。点击浏览按钮，选择本地保存路径，点击导出即可。
参数导入-可导入本地配置文件，无需进行手动设置。点击浏览按钮，选择本地导入文件，点击导入即可。
