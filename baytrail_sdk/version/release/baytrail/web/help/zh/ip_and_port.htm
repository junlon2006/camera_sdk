IP地址                               返回目录
在客户端的网络配置部分，可以对摄像机的网络参数进行设置。
IP地址设置：

1.在IP版本下拉框中选择IP版本为IPV4；

2.在模式下拉框中选择静态模式或DHCP自动获取模式，选择DHCP模式时自动获取IP，此时IP地址、子网掩码、默认网关不可设。选择静态模式，需手动设置IP地址、子网掩码、默认网关；

3.填写多播地址，即组播发送码流地址；

说明：这是一种数据包传输方式，源主机可以只需要发送一份报文就可以到达组内每个需要接收的主机上，这中间还要取决于路由器对组内和组关系的维护和选择。

4.MAC地址栏显示当前登录设备的MAC地址，不可更改；

5.设置MTU（最大传输单元）值，范围500~1500，默认为1500；

6.配置DNS，当摄像机连接以域名形式提供外部访问的平台时，请新增设置动态DNS。当模式选择静态时，DNS服务器地址需要手动填写。当模式选择DHCP时，可手动填写DNS服务器地址或者勾选自动获取DNS（注意：勾选后自动获取DNS后无法手动设置）。

端口                               
HTTP端口 HTTP通信端口，默认为80，范围为1~65535。
HTTPS端口 HTTPS通信端口，默认为5544，范围为1~65535。使用默认端口时可采用https://ip登录，使用其他端口可采用https://ip:port方式登录设备。
RTSP端口 RTSP端口号默认为554，范围为1~65535。使用RTSP登录时：rtsp://ip/id=0 (id=0播放主流，id=1播放辅流)
