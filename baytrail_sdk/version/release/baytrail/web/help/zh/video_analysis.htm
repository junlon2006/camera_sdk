移动侦测                               返回目录
侦测指定区域，检测视频的运动特征，一旦移动超出设置的灵敏度，将触发告警。告警恢复时间可在0~300s之间设置，默认5s。
设置方法：

1.勾选启用移动侦测；

2.点击开始绘制按钮，视频图像被划分成16列12行的小区域，点击画面的一个方格，以此方格为起点划出一个区域，该区域为被侦测区域，为紫红色，IPC设备可同时支持4个区域的移动侦测功能；

3.若需要重新绘制区域或清除当前绘制区域，点击清除绘制按钮即可；

4.拖动滑块调整移动侦测灵敏度；

5.点击编辑按钮，在弹出的界面中勾选时间段并设置移动侦测的开始时间和结束时间；

6.点击复制按钮，可以将设置的移动侦测时间复制到其他某一天或一整星期；

7.勾选联动方式，即移动侦测区域检测到异常情况后触发告警的联动方式。

遮挡报警                               
在监控场景中，如果摄像头的监控视野变化，被物体遮挡，不能实现正常的场景监控，摄像头遮挡检测算法能够实时检测出摄像头被遮挡，触发报警。
设置方法可参考移动侦测的设置方法。
