说明：本地存储卡首次使用时，请先点击“格式化”对磁盘进行格式化。
磁盘满策略-设置磁盘存储空间不足时的录像策略。
磁盘满策略：
覆盖 当存储空间不足时，覆盖最旧录像。默认：开启覆盖。
停止 当存储空间不足时，自动停止录像。可以在异常联动中选择磁盘满触发告警信息，提示用户本地录像已停止。
在存储设备列表中显示了所有存储设备的状态、容量、进度等信息。状态栏显示了存储设备的状态信息，如正常（即有卡且正常读写）、不存在（即没有存储卡）、未格式化（第一次插入卡需要格式化）等。进度栏以百分比显示存储卡格式化的进度。
选中磁盘，单击格式化按钮，即可对选中磁盘进行格式化。
