报警输入                               返回目录
摄像机设备通过并口接入告警输入设备（烟感器、警铃或喷淋器等）。需要分别为每个并口设置告警参数。
设置方法：

1.在报警输入号下拉框中选择报警输入号；

2.输入报警名称，即为告警输入设备命名，如前门烟感；

说明：报警名称最多输入32字符，触发告警后，名称在OSD字幕上有显示。

3.在报警类型下拉框中选择报警类型；

4.点击编辑按钮，在弹出的界面中勾选时间段并设置布防的开始时间和结束时间；

5.勾选联动方式，即在布防时间内触发告警的联动方式；

6.点击保存生效。

说明：告警输入和告警输出路数须根据具体型号确定实际支持路数，如果接入路数不够可通过告警扩展设备扩展。
告警联动方式说明：
上传中心 告警时上传监控中心。
显示字幕 告警时显示告警字幕。
声音报警 告警时产生声音报警。
录像联动 告警时录像。
抓拍 告警时抓拍图像。
告警输出 摄像机的告警输出接口，连接告警输出设备如警铃、喇叭等。告警时，所连接的输出设备发出告警声音。
PTZ联动 勾选PTZ联动，在预置点序号下拉框中选择预置点序号。告警时摄像机转向预置位。
说明：该联动方式只对带云台的摄像机有效。所选中的预置位序号需要有预置位保存。
